# Simple Calculator 🧮

This is my first project in C language, developed during my learning journey. It's a simple command-line calculator that performs basic arithmetic operations.

## ✨ Features
* **Addition (+)** and **Subtraction (-)** operations.
* Supports **floating-point numbers** (decimals).
* Simple and clean terminal interface.

## 🚀 How to Run
To compile the code, use **gcc** in your terminal:
```bash
gcc test.c -o calculator
